# 02 - 电路设计与 PCB 打板

> 目标：把面包板电路变成真正的 PCB，在嘉立创打板，自己焊接。

---

## 为什么自己做 PCB

- 🍞 面包板只能验证功能，线乱、容易松动、不永久
- 🎯 PCB 才是真正的"产品"，一次打板永久使用
- 🏭 嘉立创打板便宜（首单免费），门槛极低

## PCB 设计核心概念

### 你必须理解的几个词

| 术语 | 白话解释 |
|------|----------|
| **原理图 Schematic** | 电路的逻辑图，画"元件怎么连" |
| **封装 Footprint** | 元件在 PCB 上的"脚印"，包含焊盘尺寸和位置 |
| **PCB 布局 Layout** | 把元件放在板子上哪里 |
| **布线 Routing** | 用铜线连接各元件的焊盘 |
| **过孔 Via** | 穿到板子另一面的孔 |
| **阻焊 Solder Mask** | 板子上那层颜色（绿/蓝/紫/黑），保护铜线 |
| **丝印 Silkscreen** | 板子上的白色文字 |
| **网络 Net** | 逻辑上连在一起的一组线 |
| **Gerber 文件** | PCB 厂需要的生产文件格式 |
| **BOM** | 物料清单 Bill of Materials |
| **DRC** | 设计规则检查，防止设计错误 |

## 红绿灯 PCB 设计实战（立创 EDA）

### Step 1：新建工程

1. 打开 https://lceda.cn/
2. 新建工程 → 命名 `ai-agent-traffic-light`

### Step 2：画原理图

需要的元件：
- STM32F103C8T6 最小系统板接口（双排排母 2.54mm）
- 3x LED（0805 贴片 或 5mm 直插）
- 3x 220Ω 电阻（0805 贴片）
- 1x 100nF 电容（去耦，贴片 0805）
- 若干排针（引出 IO）

**操作流程**：
1. 放置元件 → 搜 STM32 / LED / Resistor
2. 连线 → W 键或点连线工具
3. 标注网络 → 给 GPIO 脚网络命名（LED_R, LED_Y, LED_G）
4. 加 VCC/GND 电源符号

### Step 3：检查原理图 → 转 PCB

1. 工具 → 设计规则检查（DRC）
2. 检查无误 → 设计 → 转换为 PCB

### Step 4：PCB 布局

1. 放置元件：STM32 接口居中，LED 三颗在上面一排
2. 考虑实际使用：红绿灯装在显示器上方，LED 间距对应模型位置
3. 尽量对称、美观

### Step 5：走线

- 信号线宽度：10~15mil（0.254~0.381mm）
- 电源线宽度：20~30mil
- 避免 90 度拐角（用 45 度）
- 尽量顶层走线，减少过孔

### Step 6：铺铜

- 顶层和底层都铺 GND 铜皮
- 起屏蔽和散热作用

### Step 7：DRC 检查 → 导出 Gerber

1. 运行 DRC，确认 0 错误
2. 制造 → 导出 Gerber
3. 下载 Gerber 压缩包

## 嘉立创下单流程

1. 打开 https://www.jlc.com/
2. 上传 Gerber 文件
3. 选择参数：
   - 板厚：1.6mm（常规）
   - 颜色：紫色/黑色看个人喜好
   - 表面处理：有铅喷锡（最便宜）
   - 数量：5 片（首单免费范围）
4. 确认 → 下单 → 等快递

首单 10*10cm 以内，5 片基本免费（只出邮费 10~20 元）。

## 手焊 PCB

收到板子后：

### 焊接顺序（重要！）

1. **先矮后高**：贴片电阻电容 → 排针排母 → LED → 高大的元件
2. **先贴片后直插**

### 贴片电阻焊接（0805 封装）

1. 一个焊盘上锡
2. 镊子夹电阻放到位
3. 烙铁熔化那坨锡，固定一端
4. 焊另一端
5. 回头补焊第一端

### 直插焊接

1. 元件插入，弯折引脚固定
2. 焊接
3. 剪掉多余引脚

### 焊接检查

- 万用表调到通断档（蜂鸣器）
- 测相邻焊盘是否短路
- 测电源和 GND 是否短路（**最关键的检查！**）

## AD (Altium Designer) 学习路径

如果以后想深入工业级 PCB 设计：

1. **AD 基础操作**：建工程、放置元件、连线
2. **封装库管理**：创建自己的元件库
3. **多层板设计**：4 层板叠层、内电层分割、差分线
4. **高速信号**：等长、阻抗匹配
5. **设计规范**：EMC/EMI、安规距离

**学习资源**：B站搜"Altium Designer 入门"找播放量最高的。
