## 分步实施计划

### 第 1 步：底盘 + 移动（1 周）

- STM32 控制 TB6612 驱动两个 TT 电机
- 实现前进/后退/左转/右转
- 超声波避障：检测前方距离 < 20cm → 自动停车并转向

### 第 2 步：舵机控制（1 周）

- 接入 4~6 个舵机
- 写几个预设动作（挥手、点头、旋转）
- 实现平滑的舵机运动（缓起缓停，不是直接跳到目标角度）

```c
// 舵机平滑运动
void servo_smooth_move(uint8_t channel, uint8_t target, uint16_t time_ms) {
    uint8_t current = servo_positions[channel];
    int16_t step = (target > current) ? 1 : -1;
    uint16_t delay_per_step = time_ms / abs(target - current);
    
    while (current != target) {
        current += step;
        servo_set_angle(channel, current);
        HAL_Delay(delay_per_step);
    }
}
```

### 第 3 步：ESP32 通信层（1 周）

- ESP32 联网 + MQTT 连接
- STM32 ↔ ESP32 串口协议打通
- PC 端通过 MQTT 远程控制机器人移动

### 第 4 步：AI 决策接入（1~2 周）

- ESP32 定时上传传感器数据到 PC 服务器
- PC 端 OpenCode Agent 处理数据，调用 LLM 决策
- 决策结果下发 → ESP32 → STM32 → 执行

**关键设计**：AI 不是直接控制电机，而是输出"高层指令"：
```
AI输出: "向用户方向移动" 
  → ESP32解析 → "MOVE,FORWARD,50,cm"
  → STM32执行底层控制（电机PWM+超声波避障检查）
```

### 第 5 步：语音交互（1~2 周）

- ESP32 通过 I2S 麦克风录音 → HTTP 上传 Whisper API → 文本
- 文本送 LLM 理解意图 → 决策 → 执行
- 回复文本 → TTS 合成语音 → MAX98357 播放

### 第 6 步：3D 外壳（1~2 周）

详见 Fusion 360 建模，生成 STL → 3D 打印

## 核心代码框架

### STM32 主循环

```c
typedef enum {
    STATE_IDLE,
    STATE_LISTENING,
    STATE_THINKING,
    STATE_EXECUTING,
} RobotState;

RobotState state = STATE_IDLE;
char esp_cmd[64];

int main(void) {
    HAL_Init();
    SystemClock_Config();
    MX_GPIO_Init();
    MX_USART1_UART_Init();  // 连接 ESP32
    MX_TIM2_Init();          // PWM 舵机
    MX_I2C1_Init();          // MPU6050

    while (1) {
        // 检查 ESP32 发来的指令
        if (uart_rx_ready()) {
            uart_read(esp_cmd);
            
            if (strncmp(esp_cmd, "$MOVE,", 6) == 0) {
                // 解析移动指令
                state = STATE_EXECUTING;
                robot_move(esp_cmd);
            } else if (strncmp(esp_cmd, "$GESTURE,", 9) == 0) {
                robot_gesture(esp_cmd);
            } else if (strncmp(esp_cmd, "$SPEAK,", 7) == 0) {
                // 转发给语音模块
            }
        }
        
        // 传感器采集（定时）
        if (sensor_tick()) {
            uint16_t dist = ultrasonic_read();
            MPU6050_Data imu;
            mpu6050_read(&imu);
            
            // 紧急避障（绕开AI决策，直接响应）
            if (dist < 15) {
                robot_stop();
            }
            
            // 定时上报给 ESP32
            uart_printf("$SENSORS,%d,%d,%d\r\n", dist, imu.roll, imu.pitch);
        }
    }
}
```

### ESP32 核心逻辑（伪代码）

```cpp
void loop() {
    // 1. 检查串口（STM32 发来的传感器数据）
    if (Serial2.available()) {
        String data = Serial2.readStringUntil('\n');
        if (data.startsWith("$SENSORS")) {
            // 上报给服务器
            mqtt.publish("robot/sensors", data.c_str());
        }
    }
    
    // 2. 检查 MQTT（服务器发来的指令）
    mqtt.loop();
    
    // 3. 检查语音
    if (voice_triggered()) {
        String text = voice_to_text();
        // 发给 AI 决策
        String decision = call_ai_api(text);
        // 下发 STM32
        Serial2.print("$CMD," + decision + "\r\n");
    }
}

void mqtt_callback(char* topic, byte* payload, unsigned int length) {
    String msg;
    msg.concat((char*)payload, length);
    // 转发给 STM32
    Serial2.print("$EXEC," + msg + "\r\n");
}
```

## 3D 外壳设计要点

### 软件选择

- **Fusion 360**（推荐）：参数化建模，适合机械结构
- **FreeCAD**：开源替代
- **Blender**：更适合艺术造型

### 建模技巧

1. **参数化设计**：用参数定义尺寸，方便后期修改
2. **预留空间**：电路板、舵机、电池、走线都要留余量
3. **螺丝孔**：M2 或 M3 螺丝孔，加螺母槽
4. **散热**：加通风孔
5. **分件打印**：大件切成多块，减小支撑需求
6. **公差**：FDM 打印机留 0.2~0.3mm 装配公差

### 打印设置（拓竹/其他 FDM）

- 层高：0.16~0.2mm
- 填充：15~20%（结构件可更高）
- 支撑：需要悬空处自动生成
- 材料：PLA 足够（需要耐热可用 PETG）
