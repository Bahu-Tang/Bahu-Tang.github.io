# 1 红绿灯项目：快速上手

> **目标**：当天就让红绿灯亮起来。两条路线可选。
>
> - **路线 A（速成）**：买成品红绿灯模型拆改，30 分钟见效
> - **路线 B（深造）**：从面包板接线开始，自己做全部电路

---

## 1.1 项目背景

做一个物理的"AI Agent 状态指示灯"：

- 🟢 **绿灯亮** = AI Agent 空闲，等待新任务
- 🟡 **黄灯亮** = AI Agent 工作中 / 思考中
- 🔴 **红灯亮** = AI Agent 需要人工授权 / 发生异常

挂在显示器旁边，通过 GPIO 信号控制，后续可升级为 WiFi 联网自动查询 Agent 状态。

---

## 1.2 路线 A：拆改成品红绿灯模型（速成）

> 参考飞书文档方案，适合立刻想看到效果的情况。

### 1.2.1 所需材料

| 物品 | 数量 |
|------|------|
| 红绿灯模型（淘宝 ~15元） | 1 |
| STM32 最小系统板或 ESP32 开发板 | 1 |
| 300Ω 电阻 | 3 |
| 电烙铁 + 焊锡 + 助焊膏 | 1套 |
| 杜邦线 | 若干 |
| 美工刀 | 1 |

### 1.2.2 改造步骤

#### 1.2.2.1 拆开模型

取出电池，拧下螺丝，分离前后壳。

#### 1.2.2.2 拆除原有电路

用电烙铁依次拆掉：
- **电池座**（3 个焊点）— 拆掉
- **原主控芯片**（6 脚贴片）— 烙铁宽头轻触一侧 3~5 秒，松动后轻轻推动即可取下
- **两个电阻**（标"R"，上面印"300"）— 拆掉
- **一个电容**（标"C"）— 拆掉

> ⚠️ 所有焊点拆完后用吸锡器 + 助焊膏清理干净，防止短路。

#### 1.2.2.3 刮焊盘引出 LED 负极

用美工刀轻刮**上方绿灯负极、下方红灯负极**引脚下方绿油，露出黄铜色。
- 绿/红灯珠有 "+" "-" 标记（实在看不清问 AI 看图）
- 中间黄灯可复用下方 LED 负极焊盘

> ⚠️ 轻刮！别刮断铜箔。

#### 1.2.2.4 焊接限流电阻

3 个 300Ω 左右（200~800Ω 都行，越小越亮但别太小烧 LED）电阻：
- 电阻一端焊到刚刮出的铜箔
- 另一端弯折引线到板侧出线

> 💡 电阻引脚弯折要轻，焊盘很容易脱落！焊好后用热熔胶/绿油/502 加固。

#### 1.2.2.5 穿线打孔

后壳底部打孔（电烙铁烫 or 小钻头），连接线从中穿入。接线可以在背后焊：
- **GPIO G0** → 绿灯负极引出的电阻
- **GPIO G1** → 黄灯负极引出的电阻
- **GPIO G2** → 红灯负极引出的电阻
- **3V3** → 正极（原电池座左右焊点）→ 保留原开关控制

#### 1.2.2.6 固定装回

热熔胶固定电阻、接线 → 装回外壳 → 拧螺丝 → 装回面罩。

#### 1.2.2.7 驱动板处理

电工胶布缠一下裸露的 GPIO 板就行，或用热缩管套上。

#### 1.2.2.8 挂到显示器

3M 亚克力双面胶（有厚度的）粘在显示器边框即可。

## 1.3 路线 B：面包板 DIY（深造）

> 从零开始，用三颗独立 LED + 电阻 + 面包板搭建。

### 1.3.1 电路计算：LED 限流电阻

不同颜色 LED 的正向压降不同：

| LED 颜色 | 正向压降 Vf | 工作电流 If | GPIO 3.3V 下电阻 |
|----------|------------|------------|-------------------|
| 🟢 绿色 | ~2.2V | 5~10mA | (3.3-2.2)/0.01 ≈ 110Ω → 选 100~220Ω |
| 🟡 黄色 | ~2.0V | 5~10mA | (3.3-2.0)/0.01 ≈ 130Ω → 选 100~220Ω |
| 🔴 红色 | ~1.8V | 5~10mA | (3.3-1.8)/0.01 ≈ 150Ω → 选 150~220Ω |

**公式**：R = (V_supply - Vf) / If

> 用 5V 供电（如 Arduino）时：红 (5-1.8)/0.01 = 320Ω → 选 220~470Ω
> 飞书文档说的"300欧左右"是通用安全值，3.3V 和 5V 都适用。

### 1.3.2 接线图（面包板）

```
STM32 最小系统板
┌─────────────────┐
│           GND ───┼──┬─[R_红]─┬─[LED红]─┐
│           PA0 ───┼──┘         │         │
│           PA1 ───┼──┬─[R_黄]─┬─[LED黄]─┤── GND
│           PA2 ───┼──┘         │         │
│          3V3  ───┼──┬─[R_绿]─┬─[LED绿]─┘
│                 │  └─────────┘
└─────────────────┘

所有 LED 阳极接 GPIO（通过电阻），阴极共地
```

**每路接线**：GPIO → 电阻 → LED 阳极(+) → LED 阴极(-) → GND

> ⚠️ 电阻接在阳极还是阴极都行，但建议统一接阳极侧，好记。

### 1.3.3 核对清单

- [ ] LED 长脚(+) → 电阻 → GPIO，短脚(-) → GND
- [ ] 电阻值在 100~470Ω 之间
- [ ] 面包板插紧，不松动
- [ ] STM32 板通过 USB 供电

## 1.4 固件代码

### 1.4.1 STM32 (STM32CubeIDE / HAL 库)

```c
// main.c 核心部分 - 红绿灯交替闪烁 + 单灯控制
#include "main.h"

void set_light(uint8_t color) {
    // PA0=红 PA1=黄 PA2=绿  (根据实际接线改)
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_0, GPIO_PIN_RESET);
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_1, GPIO_PIN_RESET);
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_2, GPIO_PIN_RESET);
    
    switch(color) {
        case 0: HAL_GPIO_WritePin(GPIOA, GPIO_PIN_0, GPIO_PIN_SET); break; // 红
        case 1: HAL_GPIO_WritePin(GPIOA, GPIO_PIN_1, GPIO_PIN_SET); break; // 黄
        case 2: HAL_GPIO_WritePin(GPIOA, GPIO_PIN_2, GPIO_PIN_SET); break; // 绿
    }
}

int main(void) {
    HAL_Init();
    SystemClock_Config();
    // 初始化 PA0, PA1, PA2 为输出
    MX_GPIO_Init();

    while (1) {
        // 演示：红→黄→绿 循环
        set_light(2);  HAL_Delay(2000); // 绿灯 2秒
        set_light(1);  HAL_Delay(500);  // 黄灯 0.5秒
        set_light(0);  HAL_Delay(2000); // 红灯 2秒
    }
}
```

### 1.4.2 ESP32 / Arduino (PlatformIO)

```cpp
#define LED_RED   13
#define LED_YELLOW 12
#define LED_GREEN 14

void setup() {
    pinMode(LED_RED, OUTPUT);
    pinMode(LED_YELLOW, OUTPUT);
    pinMode(LED_GREEN, OUTPUT);
}

void setLight(int color) {
    digitalWrite(LED_RED, LOW);
    digitalWrite(LED_YELLOW, LOW);
    digitalWrite(LED_GREEN, LOW);
    
    switch(color) {
        case 0: digitalWrite(LED_RED, HIGH); break;
        case 1: digitalWrite(LED_YELLOW, HIGH); break;
        case 2: digitalWrite(LED_GREEN, HIGH); break;
    }
}

void loop() {
    setLight(2); delay(2000); // 绿
    setLight(1); delay(500);  // 黄
    setLight(0); delay(2000); // 红
}
```

## 1.5 烧录固件

### 1.5.1 ST-Link 烧录（推荐）

1. 连接 ST-Link 到 STM32 板（SWDIO/SWCLK/GND/3.3V）
2. STM32CubeIDE → Run → 或 STM32CubeProgrammer → Connect → Erase & Program

### 1.5.2 串口烧录（备选）

1. USB-TTL 接 STM32（RX/TX/GND/3.3V）
2. STM32 BOOT0 跳线置 1，按 RESET
3. STM32CubeProgrammer → UART → Connect → Download

## 1.6 常见问题

| 问题 | 排查 |
|------|------|
| LED 不亮 | 方向是否反了？长脚(+)接 GPIO，短脚(-)接 GND |
| LED 很暗 | 电阻太大，减小到 100~220Ω |
| LED 烧了 | 电阻太小或没接电阻，GPIO 直连 LED 会烧 |
| GPIO 无输出 | 检查 GPIO 初始化代码，是否配置为推挽输出 |
| 焊盘脱落 | 绿油/502/AB 胶修复，或用导线飞线 |
| ESP32 GPIO 5V 输出电压 | ESP32 IO 是 3.3V 不是 5V，放心用。5V 引脚仅供电源 |
