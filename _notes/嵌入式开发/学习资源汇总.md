## 1.1 🎬 B站 视频教程

### 1.1.1 STM32 入门

- **[野火] STM32 教程** — 经典中的经典，从寄存器讲到 HAL 库
  - https://www.bilibili.com/video/BV1Vx411r7WD
- **[正点原子] STM32 教程** — 资料非常完善，有配套书籍和开发板
  - https://www.bilibili.com/video/BV1kx411k7JT
- **[江协科技] STM32 入门** — 比较精简，适合快速入门
  - https://www.bilibili.com/video/BV1th411z7sn

### 1.1.2 电路设计 / PCB

- **[立创 EDA] 官方教程** — 从原理图到 PCB 一条龙
  - 在立创 EDA 网页里就有教程入口
- **[KiCad] 入门到精通** — B站搜"KiCad 教程"
- **[嘉立创] 打板流程** — B站搜"嘉立创打板 新手"

### 1.1.3 Arduino / ESP32

- **[太极创客] ESP32 教程** — 非常详细
  - https://www.bilibili.com/video/BV1tv411w7vd
- **[Arduino 入门]** — 官方中文文档也可以看

### 1.1.4 焊接

- B站搜"手工焊接教程 贴片" — 找播放量高的看就行

## 1.2 📚 文档 / 书籍

### 1.2.1 官方文档

- **STM32 HAL 库手册**：STM32CubeIDE 安装后自带
- **STM32F103 参考手册**：STM32F103C8 数据手册，必读
  - https://www.st.com/resource/en/reference_manual/cd00171190.pdf
- **ESP32 技术参考手册**
  - https://www.espressif.com/zh-hans/support/documents/technical-documents

### 1.2.2 推荐书籍（按需选读）

| 书名 | 说明 |
|------|------|
| 《STM32 库开发实战指南》刘火良 | 野火配套，实战导向 |
| 《Altium Designer PCB 画板速成》 | 想学 AD 的话看这本 |
| 《电子设计从零开始》 | 模拟电路入门，适合零基础 |
| 《爱上单片机》杜洋 | 轻松有趣，适合培养兴趣 |

## 1.6 📝 遇到问题怎么办

1. **先问 AI**（ChatGPT / Claude / OpenCode）— 描述清楚你做了什么、什么现象、错误信息
2. **B 站搜** — 很多问题有视频演示
3. **立创社区 / 51黑论坛** — 发帖求助，电路图+实物图+错误描述 三件套
