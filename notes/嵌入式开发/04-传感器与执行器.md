## 舵机（Servo）

### 原理

舵机通过 PWM 信号的高电平脉宽控制角度：

| 脉宽 | 角度 | 说明 |
|------|------|------|
| 500μs | 0° | 最左 |
| 1000μs | 45° | |
| 1500μs | 90° | 中间 |
| 2000μs | 135° | |
| 2500μs | 180° | 最右 |

**信号周期**：20ms（50Hz）

> ⚠️ 注意：部分舵机只有 0~90° 或 0~270° 范围，看规格书。

### PWM 配置（STM32）

```c
// 定时器配置：50Hz → ARR=19999, PSC=71 (72MHz)
// 角度转脉宽函数
void servo_set_angle(uint8_t angle) {
    // angle: 0~180
    // 映射到 500~2500 (即 0.5ms~2.5ms)
    uint16_t pulse = 500 + (angle * 2000 / 180);
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, pulse);
}
```

### 供电注意

- SG90 微舵：5V 直接从 STM32 板子的 5V 引脚取电（单颗）
- **多颗舵机**：必须用独立电源！STM32 的 3.3V/5V 带不动多个舵机
- 舵机电源和控制信号共地（GND 连在一起）

## 直流电机

### 不能直接接 GPIO！

电机电流大（几百mA~几A），GPIO 只能输出 ~20mA，直接接必烧。

### 驱动方案

**TB6612（推荐）**：双路电机驱动，体积小，发热小

接线：
```
STM32       TB6612        电机
PA6  ────  PWMA
PA4  ────  AIN1
PA5  ────  AIN2         AO1 ──── 电机+
                        AO2 ──── 电机-
VM   ────  12V 电源
GND  ────  共地
```

**L298N（备选）**：便宜但体积大、压降大（损失 ~2V）

### 控制逻辑

| AIN1 | AIN2 | 效果 |
|------|------|------|
| 0 | 0 | 刹车 |
| 1 | 0 | 正转 |
| 0 | 1 | 反转 |
| 1 | 1 | 刹车 |

PWM 接 PWMA 端来调速。

```c
void motor_forward(uint8_t speed) {
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_4, GPIO_PIN_SET);  // AIN1=1
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_5, GPIO_PIN_RESET); // AIN2=0
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, speed);  // PWM调速
}
```

## 常用传感器

### HC-SR04 超声波测距

**原理**：发 40kHz 超声波脉冲 → 计时回波时间 → 距离 = (时间 × 声速) / 2

接线：Trig（触发）、Echo（回波）、VCC、GND

```c
float ultrasonic_distance(void) {
    // 1. 发送 10μs 高脉冲到 Trig
    HAL_GPIO_WritePin(TRIG_PORT, TRIG_PIN, GPIO_PIN_SET);
    delay_us(10);
    HAL_GPIO_WritePin(TRIG_PORT, TRIG_PIN, GPIO_PIN_RESET);
    
    // 2. 等待 Echo 变高，开始计时
    while(HAL_GPIO_ReadPin(ECHO_PORT, ECHO_PIN) == 0);
    uint32_t start = micros();
    
    // 3. 等待 Echo 变低，停止计时
    while(HAL_GPIO_ReadPin(ECHO_PORT, ECHO_PIN) == 1);
    uint32_t end = micros();
    
    // 4. 计算距离 (cm)
    float distance = (end - start) * 0.034 / 2.0;
    return distance;
}
```

### MPU6050（六轴陀螺仪+加速度计）

通过 **I2C** 通信，获取姿态角。

```c
// 初始化
mpu6050_init(&hi2c1);

// 读取数据
MPU6050_Data data;
mpu6050_read(&hi2c1, &data);
// data.accel_x, accel_y, accel_z  (加速度)
// data.gyro_x, gyro_y, gyro_z    (角速度)
// data.temp                        (温度)
// data.pitch, data.roll           (姿态角，需要姿态解算)
```

> 完整的姿态解算代码较长，建议让 AI 帮你补全。

### 红外避障模块

最简单的数字传感器：前方有障碍 → OUT 输出低电平。

```c
uint8_t obstacle = HAL_GPIO_ReadPin(GPIOB, GPIO_PIN_3);
if (obstacle == 0) {
    // 有障碍物，停车/转向
}
```

## 动手实践：简易舵机云台

### 材料

- SG90 舵机 ×2
- MPU6050 ×1
- 面包板 + 杜邦线

### 功能

- 一个舵机控制水平旋转（Pan），一个控制俯仰（Tilt）
- 读取 MPU6050 姿态数据，让云台保持水平（简单自稳）

### 代码框架

```c
// 初始化
servo_init();     // 配置两个 PWM 通道
mpu6050_init();   // 配置 I2C

while(1) {
    MPU6050_Data data;
    mpu6050_read(&hi2c1, &data);
    
    // 用 Roll 和 Pitch 修正舵机角度
    int16_t pan_angle  = 90 + data.roll  * 0.1;  // 系数需要调
    int16_t tilt_angle = 90 + data.pitch * 0.1;
    
    // 限幅
    if (pan_angle < 0) pan_angle = 0;
    if (pan_angle > 180) pan_angle = 180;
    
    servo_set_angle(SERVO_PAN, pan_angle);
    servo_set_angle(SERVO_TILT, tilt_angle);
    
    HAL_Delay(20); // 50Hz 更新率
}
```

## 供电设计要点

### 多电压系统

```
12V DC 输入（电源适配器）
  ├── → 舵机 12V（大电流）
  ├── → LM2596 降压 → 5V（传感器、ESP32）
  └── → AMS1117-3.3 → 3.3V（STM32 MCU）
```

### 共地原则

**所有模块的 GND 必须连在一起**，包括：
- 主控板 GND
- 舵机电源 GND
- 传感器 GND
- ESP32 GND

不共地信号无法通信。
