# 1 工具链环境搭建

> 涵盖 WSL2 / Windows 下所有开发工具的安装和配置。

---

## 1.1 固件开发环境

### 1.1.1 STM32CubeIDE（推荐新手）

STM32 官方 IDE，集成 CubeMX（图形化配置外设）+ GCC 编译 + 调试器，开箱即用。

- **下载**：https://www.st.com/en/development-tools/stm32cubeide.html
- **安装**：一路 Next，建议安装在 SSD 上
- **注意**：需要注册 ST 账号（免费）

**优点**：
- 图形化配置引脚和外设（CubeMX），不用背寄存器
- HAL 库代码自动生成
- 集成编译和调试

**缺点**：
- 基于 Eclipse，界面较重
- 不如 VSCode 灵活

### 1.1.2 PlatformIO（推荐有编程经验者）

基于 VSCode 的插件，支持 STM32、ESP32、Arduino 等数百种平台。

**安装步骤**：
1. 安装 VSCode
2. 在扩展商店搜索 `PlatformIO IDE` 安装
3. 等待 PlatformIO 自动下载工具链

**新建 STM32 项目**：
```
PlatformIO Home → New Project → Board: STM32F103C8 (Blue Pill) → Framework: STM32Cube
```

**优点**：
- 轻量，基于 VSCode
- 统一的项目管理
- 库管理方便

**缺点**：
- 不能像 CubeIDE 那样图形化配置引脚（需手动写代码或用 STM32CubeMX 生成后导入）

### 1.1.3 烧录工具

**ST-Link Utility**（ST 官方）：
- 下载：https://www.st.com/en/development-tools/stsw-link004.html
- 用于通过 ST-Link 调试器烧录固件

**STM32CubeProgrammer**（ST 官方，推荐）：
- 下载：https://www.st.com/en/development-tools/stm32cubeprog.html
- 支持 ST-Link、USB-TTL 串口多种烧录方式

**OpenOCD**（开源）：
- PlatformIO 自带，命令行使用
- 高级用户可配合 GDB 调试

## 1.2 电路设计 EDA 工具

### 1.2.1 立创 EDA（推荐入门）

国产在线 EDA，直接对接嘉立创打板和元器件商城。

- **入口**：https://lceda.cn/
- **模式**：专业版（功能更全）和标准版（更简单）
- **用法**：浏览器直接使用，无需安装

**核心优势**：
- 元件库和嘉立创商城直接打通，选元件就能看到价格和库存
- 一键导出 Gerber 下单打板
- 免费、中文、教程多

### 1.2.2 KiCad（推荐进阶）

开源专业 EDA，功能对标商业软件，完全免费。

- **下载**：https://www.kicad.org/
- **安装**：标准安装包，Windows/Mac/Linux 都支持

**优点**：
- 无任何使用限制
- 插件生态丰富
- 业界认可度高

**缺点**：
- 学习曲线比立创 EDA 陡
- 元件库需要自己维护

### 1.2.3 Altium Designer（行业标准）

商业 EDA 软件，企业级功能，PCB 设计行业标杆。

- **获取**：需要付费 License（学生可申请教育版）
- **适合**：有条件的深入学习，了解工业级 PCB 设计流程

**建议**：先立创 EDA 入门，掌握原理图/PCB 基本概念后，再学 KiCad 或 AD。

## 1.3 3D 建模工具

### 1.3.1 Fusion 360（推荐）

- **获取**：个人/教育版免费
- **下载**：https://www.autodesk.com/products/fusion-360/personal
- **适合**：建模、装配、生成工程图

### 1.3.2 FreeCAD（开源替代）

- **下载**：https://www.freecad.org/
- **适合**：不想依赖商业软件的替代方案

### 1.3.3 Blender

- **下载**：https://www.blender.org/
- **适合**：更偏艺术建模，不适合参数化工程建模

---

## 1.4 串口调试工具

- **MobaXterm**（Windows，推荐）：功能强大的终端，带串口功能
- **PuTTY**（Windows）：经典串口工具
- **Minicom**（Linux WSL）：`sudo apt install minicom`
- **Arduino IDE 串口监视器**：最简单，装一个当串口工具用也行

---

## 1.5 其他实用工具

- **嘉立创下单助手**：https://www.jlc.com/ 手机版/电脑版
- **ESP32 Flash Download Tool**：ESP32 专用烧录
- **Fritzing**：面包板接线图绘制（入门友好）
- **Wokwi**（在线仿真，推荐）：https://wokwi.com/ 无需硬件就能模拟 Arduino/ESP32 电路

## 1.6 安装优先级

| 优先级 | 工具 | 用途 |
|--------|------|------|
| 🔴 立即安装 | STM32CubeIDE + STM32CubeProgrammer | 写代码+烧录 |
| 🔴 立即安装 | 立创 EDA（浏览器） | 电路设计 |
| 🟡 后续安装 | VSCode + PlatformIO | 替代 IDE |
| 🟡 后续安装 | KiCad | 进阶 EDA |
| 🟢 按需安装 | Fusion 360 / FreeCAD | 3D 建模 |
| 🟢 按需安装 | Wokwi（网页） | 在线仿真 |
